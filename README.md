# Simple Tabs

A lightweight, interactive tabs component built with vanilla HTML, CSS, and JavaScript. Click on the tab buttons to switch between different content sections.

## How to Use

1. Open `html` in your web browser
2. Click on any tab button to view its content
3. The active tab is highlighted automatically
