# Simple Tabs

A lightweight, interactive tabs component built with vanilla HTML, CSS, and JavaScript. Click on the tab buttons to switch between different content sections. Includes a basic cookie consent banner for user privacy.

## Features

- Interactive tab switching with smooth content display
- Highlighted active tab
- Cookie consent banner that appears on first visit (accepts cookies for 1 year)

## How to Use

1. Open `html` in your web browser
2. Click on any tab button to view its content
3. Accept the cookie consent banner if prompted (it won't reappear after acceptance)

## Files

- `html`: Main HTML structure
- `style.css`: Styling for tabs and consent banner
- `script.js`: JavaScript for tab functionality and cookie consent logic
